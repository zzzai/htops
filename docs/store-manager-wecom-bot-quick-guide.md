# 店长企微问数速查卡

这份只讲 3 件事：

1. 发什么话
2. 什么时候发
3. 查完后怎么做

## 第一次先问

直接发：

```
我现在能查哪家店
```

你要先确认 2 件事：

- 你有没有权限
- 你绑定的是哪家店

如果机器人回复你没有权限，就先找管理员。

## 最常用 4 句

### 1. 看昨日日报

直接发：

```
昨天日报
```

如果你管多家店，就写清门店：

```
迎宾店昨天日报
```

### 2. 只查几个指标

直接发：

```
迎宾店四月十九日点钟率和加钟率
```

也可以发：

```
迎宾店四月十九日营收和客流
```

### 3. 问今天先抓什么

直接发：

```
迎宾店昨天有什么风险和建议
```

如果你只管 1 家店，也可以直接发：

```
昨天有什么风险和建议
```

## 什么时候发

### 早上

先发：

```
昨天日报
```

再发：

```
昨天有什么风险和建议
```

### 中午

如果你还没看懂昨天问题在哪，就发：

```
迎宾店昨天有什么风险和建议
```

### 晚上

只查你今天盯的那个指标。

比如今天盯点钟率，就发：

```
迎宾店今天点钟率
```

## 群里怎么发

先点一下机器人，再发问题。

比如：

```
迎宾店昨天营收
```

问话尽量带 3 样：

1. 门店
2. 时间
3. 指标

比如：

```
迎宾店昨天营收
华美店四月十九日日报
义乌店昨天点钟率
```

## 查完后怎么做

### 客流掉了

先做：

- 看进店承接
- 看晚场

再问：

```
迎宾店近七天客流为什么掉了
```

### 加钟率掉了

先做：

- 盯服务结束前推荐

再问：

```
迎宾店今天加钟率
```

### 点钟率掉了

先做：

- 盯熟客
- 盯指定技师

再问：

```
迎宾店今天点钟率
```

### 新增会员、储值弱

先做：

- 先看昨天风险和建议

再问：

```
迎宾店昨天有什么风险和建议
```

## 不要这样问

不要发：

```
昨天怎么样
```

改成：

```
迎宾店昨天营收
```

不要一次问很多件事：

```
这家店为什么不行，谁的问题，怎么改，名单也给我
```

改成分开问：

```
迎宾店近七天营收下滑原因
迎宾店四月十九日点钟率和加钟率
迎宾店昨天有什么风险和建议
```

## 店长先记住这 4 类就够了

1. 盘子有没有问题
2. 近7天有没有风险
3. 该先抓什么
4. 营收、点钟率、钟效
